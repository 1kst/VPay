Vpay —— 个人开发者收款解决方案

注意
Vpay3.0相较于Vpay2.0有着极大的改变，请务必删除原有的网站重新配置。

Vpay 是基于cleanphp + mysql + 支付宝当面付 实现的一套支付程序，主要包含以下特色：

  1. 官方收款渠道，回调更快
  2. 免费、开源，无后门风险
  3. Vpay仅供个人开发者调试测试使用，请勿用于非法用途。
